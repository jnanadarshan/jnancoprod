    info About ESPHome Display Designer

    Key Features

    • Support for text, icons, shapes, and bars. Triangles - coming soon
    • Live preview with Material Icons
    • Export to ESPHome-compatible code
    • Export and import designs

    Known Limitations

    • Font previews are approximate
    • All Material Icons will not appear in search window.
    • Need search to Icon Names from https://fonts.google.com/icons manually
    • Corner radius is preview-only {ESPHome limitation}
    • Some Material Icons may not render correctly
    • UI does not render properly on a Mobile Device>

    Known Bugs

    • Import TXT file feature has issues
    • On page reload need to click on elements name for Icons to display correctly
    • Screen Resolution beyond 480px overflows in UI and scaling button has to be used.
